动态类型

发表于 2017-12-23 | 阅读数
字数统计: 335 | 阅读时长 ≈ 1

Python是动态语言类型。相对于静态语言类型，动态语言类型意味着变量没有一个固定的类型。变量被实现为指向对象的指针，因此可以先让变量指向一个数字，再指向一个函数，再指向一个字符串。

避免为不同的事物使用相同的变量名，下面是一些建议和例子：

建议 坏的例子 好的例子
使用短小函数或者方法，以降低不相关事物同名的风险 a = 1
a = “answer is {}”.format(a)		 def get_answer(a):
return “answer is {}”.format(a)
a = get_answer(1)
相关事物，类型不同，也应使用不同的名称 # 一个字符串
items = “a b c”
# 一个列表
items = items.split(“ “)		 items_str = “a b c”
items_list = items.split(“ “)

复用变量名对于提升代码效率毫无帮助：赋值时同样会创建一个新的对象。随着代码越来越复杂，在同一个变量名多次赋值之间被其他代码行分隔开，将导致确定给定变量的类型更加困难。

在一些实践中，比如函数式编程，建议不要对同一变量进行多次赋值。如果对同一个变量，重复赋值为不同的类型，Pylint会输出警告提醒你。
